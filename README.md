# TRAPPIST System Pipeline

One-page BBC Pipeline R&D document for the four-shot TRAPPIST-1 Unreal Engine to Seedance workflow.

## Open

Open `index.html` in a browser.

## Structure

- `Hero`: TRAPPIST-1 system pipeline title and BBC Pipeline R&D identity.
- `First Cut`: large featured video for the first cut of the four-shot sequence.
- `How It Was Done`: NASA-backed data, Unreal layout, and scale strategy.
- `Look Development`: planet/red-dwarf reference-prompt-output rows, followed by four playblast placeholders over four final-frame placeholders.
- `Cinematic Look Development`: playblast, first frame, and final AI render.
- `Prompt Template`: TRAPPIST video instruction block.
- `Closing Note`: controlled hybrid pipeline summary.
